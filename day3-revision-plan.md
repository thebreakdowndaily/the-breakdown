## LOGICAL REASONING (6-8 Qs | Moderate-Hard | PRACTICE = SCORE)

### Syllogisms — The 100% PYQ Topic

**Rules for "Only A are B" type:**
- "All A are B" = A inside B circle
- "Only A are B" = All B are A (reverse!)
- "No A is B" = Separate circles
- "Some A are B" = Overlapping circles

**Venn Diagram Method (3 statements):**
Draw 3 overlapping circles → shade what's given → check what MUST be true

**Opposition of Square (NTA LOVES THIS):**

| Pair | Relation | Truth Value |
|------|----------|-------------|
| A (All S are P) ↔ E (No S is P) | Contrary | Both cannot be true, both can be false |
| A (All S are P) ↔ O (Some S are not P) | Contradictory | One true, other MUST be false |
| E (No S is P) ↔ I (Some S are P) | Contradictory | One true, other MUST be false |
| I (Some S are P) ↔ O (Some S are not P) | Sub-contrary | Both can be true, both cannot be false |

**Memory Trick:** A-E = both can't be true / I-O = both can't be false / A-O & E-I = MUST disagree

### Fallacies (PYQ Regular)

| Fallacy | Definition | Example |
|---------|------------|---------|
| Ad Hominem | Attack the person, not argument | "He's uneducated, so his point is wrong" |
| Straw Man | Misrepresent argument to attack it | "You want regulation? So you want communism?" |
| Appeal to Authority | "Expert said so" = must be true | "This doctor says vaccines cause X" (out of field) |
| Circular Reasoning | Premise assumes conclusion | "Bible is true because God wrote it" |
| False Dilemma | Only two options when more exist | "Either with us or against us" |
| Slippery Slope | One step leads to extreme outcome | "Marijuana leads to heroin addiction" |
| Hasty Generalization | Small sample → broad conclusion | "My uncle smoked and lived to 90" |
| Post Hoc Ergo Propter Hoc | After this, therefore because of this | "I wore blue socks and India won → blue socks are lucky" |

## ADVERTISING & PUBLIC RELATIONS (8-10 Qs | Moderate)

### Advertising Theories/Models

| Model | Full Name | Steps |
|-------|-----------|-------|
| AIDA | Attention → Interest → Desire → Action | Classic, most asked |

**Mnemonic:** AIDA = "Attention! I Desire Action"

### PR Models (Grunig & Hunt — 1984)

| Model | Direction | Nature |
|-------|-----------|--------|

| Public Information | One-way (org → public) | Factual dissemination |
| Two-Way Asymmetric | Two-way, unbalanced | Persuasion, feedback used to benefit org |
| Two-Way Symmetric | Two-way, balanced | Mutual understanding, dialogue |

**PYQ Pattern:** Match PR model with description. Two-Way Symmetric = ideal.

## ERROR ANALYSIS — COMMON MISTAKES

### Top 10 Mistakes Students Make

| Mistake | Why It Happens | How to Fix |
|---------|---------------|------------|
| Confusing Stratified vs Cluster sampling | Both involve "groups" | Stratified = random from EACH group / Cluster = random ENTIRE groups |
| Misreading Assertion-Reason codes | Confusing (a) and (b) | Cover the options, decide if R explains A first |
| Forgetting "Only A are B" = "All B are A" | Language trap | Draw Venn diagram |
| Confusing Reliability vs Validity | Both sound similar | Reliability = repeatable / Validity = accurate |
| Thinking Teaching Aptitude is "common sense" | Familiar topic → underprepared | NTA asks SPECIFIC models and scholars |
| Mixing up journalist years/chronology | Too many dates | Focus on TOP 10 newspapers only |
| Guessing on Indian Logic terms | Unfamiliar vocabulary | MEMORIZE: Vyapti, Hetu, Hetvabhas, Anumana |
| Confusing Normative Theories | Similar names | Libertarian = freedom / Social Resp = freedom + obligation |
| Ignoring ICT abbreviations | "Too technical" | 15 min memorization = 5 marks guaranteed |
| Losing time on DI calculations | Overthinking | Skip hard calculations, return later |
