📐 Design Specs

How to recreate these in Canva or any design tool

Canvas Size

1080×1920 pixels
9:16 aspect ratio
Set as custom size in Canva / Photoshop / Figma

Typography

Titles: Bebas Neue, 32-48px, letter-spacing 2-4px
Body: Source Sans 3, 14-18px, weight 400
Labels: Source Sans 3, 10-12px, uppercase

Colors

BG: #0f0f1a / #1a1a2e
Gold: #e4a11d
Text: #ffffff / #888888
Cards: #16213e

Elements

• Split-B logo (bottom-right, faded)
• Gold accent line (center or under title)
• "THE BREAKDOWN" top-left label
• 8-12px padding from edges

Gradients

Use subtle gradient overlays to differentiate categories:
• Standard: 135deg #0f0f1a → #1a1a2e
• Urgent: Add #1a0a0a (red tint)
• Growth: Add #0d1a0d (green tint)

Pro Tips

• Batch-create 1 week of stories in 30 min
• Save each template as a Canva preset
• Keep text above the "safe zone" (top/bottom 10%)
• Leave room for Instagram stickers (top 15%)
