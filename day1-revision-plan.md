# UGC NET / JRF — DAY 1 REVISION PLAN

## PAPER I: TEACHING APTITUDE + RESEARCH APTITUDE

### TEACHING APTITUDE (5-8 Qs / Easy-Moderate / MUST SCORE)

#### Levels of Teaching

| Level | Focus | Teacher Role | Learner Role |
|-------|-------|-------------|--------------|
| Memory | Rote recall | Transmitter | Passive receiver |
| Understanding | Comprehension + interpretation | Explainer | Active participant |
| Reflective | Critical thinking + application | Facilitator | Problem-solver |

NTA loves: Understanding Level questions. Memory Level = least desirable. Reflective = JRF-level.

#### Bloom's Taxonomy (Revised — Anderson & Krathwohl 2001)

Mnemonic: R U A A E C

| Level | Verbs | NTA Pattern |
|-------|-------|-------------|
| Remember | Define, list, recall | Match the following |
| Understand | Explain, interpret | Statement-based |
| Apply | Demonstrate, solve | Scenario-based |
| Analyze | Compare, differentiate | Assertion-Reason |
| Evaluate | Critique, justify | JRF analytical |
| Create | Design, construct | Rare in MCQs |

#### Evaluation: Formative vs Summative

| Formative | Summative |
|-----------|-----------|
| During learning | After learning |
| Continuous | Terminal |
| Feedback-oriented | Grade-oriented |
| Quizzes, classwork | Final exam |

#### Teaching Methods

| Inductive | Deductive |
|-----------|-----------|
| Specific to General | General to Specific |
| Examples to Rule | Rule to Examples |
| Discovery-based | Lecture-based |

#### Scholars: Piaget (stages), Vygotsky (ZPD), Skinner (operant conditioning), Maslow (hierarchy)

### RESEARCH APTITUDE (6-8 Qs / Moderate / HIGH SCORE)

#### Types of Research

| Type | Purpose |
|------|---------|
| Basic/Pure | Knowledge for its own sake |
| Applied | Solve specific problem |
| Action | Solve classroom/org problem |
| Evaluative | Assess program effectiveness |

Qualitative (words/themes) vs Quantitative (numbers/stats)

#### Sampling Techniques

Probability (PYQ FAVOURITE):

Mnemonic: S S S C

| Method | Description |
|--------|-------------|
| Simple Random | Lottery method |
| Stratified | Divide into strata, random from each |
| Systematic | Every nth element |
| Cluster | Randomly select groups |

Non-Probability:

Mnemonic: C P S Q

| Method | Description |
|--------|-------------|
| Convenience | Whoever is available |
| Purposive | Researcher's judgment |
| Snowball | Participants refer others |
| Quota | Fill predetermined categories |

#### Hypothesis

| Null (H₀) | Alternative (H₁) |
|-----------|-----------------|
| No relationship | Relationship exists |
| What you try to reject | What you try to prove |

#### Research Ethics (NTA's NEW focus 2023+)

- Plagiarism types: Copy-paste, Mosaic, Self-plagiarism
- Informed consent is mandatory
- Anonymity does NOT equal Confidentiality
- IRB approval needed

#### Reliability vs Validity

| Reliability | Validity |
|-------------|----------|
| Consistency | Accuracy |
| Same result every time | Measures what it claims |
