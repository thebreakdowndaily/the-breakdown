Usage rules: Left rect = solid. Right rects = split (top accent, bottom neutral). Minimum clear space = 1x logo width. Never stretch, rotate, or add effects.

Rule: Every carousel starts with a dark background (#0A0A0A) and gold accent. Stats always use gold text on dark. CTAs always use dark text on gold. This creates instant brand recognition in the feed.

Typography System

Body / Context Text

Used for: Context slides, captions, body copy

Data / Technical
JetBrains Mono
Used for: Labels, percentages, hex codes, tags
Size: 9-11px (slides), 10-12px (reels)
Type Rules:

• Slide 1: Max 12 words. Space Grotesk 24-28px. White on black.
• Context slides: Max 40 words. Inter 14px. Left-aligned.
• Stats: Number = Space Grotesk 48-56px gold. Label = Inter 14px gray.
• Labels: Always JetBrains Mono 9-11px, uppercase, gold, 2-3px letter-spacing.
• CTAs: Always dark text on gold background.

Hierarchy:
1. Hook (slide 1) — loudest
2. Stat/Contrarian — emotional peak
3. Context — informational
4. CTA — action
5. Brand — subtle

08 — Hashtag Strategy

Tiered Hashtag System

Tier 1 — Reach (100K+ posts)
#India #News #AI #Technology #Business #StartupIndia #Education #FutureOfWork
5-8 per post. Broad discovery.
Tier 2 — Niche (10K-100K posts)
#FactCheck #DigitalMarketing #CreatorEconomy #StartupIndia #CareerAdvice #SkillIndia #Deepfake
5-8 per post. Targeted audience.
Tier 3 — Trending (Daily rotation)
Rotate based on daily trending topics. Check Instagram Explore every morning.
3-5 per post. Ride the wave.
Tier 4 — Brand (Every post)
#TheBreakdown #Newsjack #TheBreakdownDaily
2-3 per post. Always. Builds brand hashtag equity.
Rule: 20-25 hashtags per post. Mix all 4 tiers. Never use the same set twice in a row. Instagram penalizes hashtag spam — vary your mix.
