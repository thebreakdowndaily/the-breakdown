# THE BREAKDOWN — Homepage Generator

Generate or update the homepage with dynamic sections.

## Homepage Sections

### Breaking News Ticker
Latest urgent stories, scrolling

### Featured Story
Hero layout — full-width with large image

### Trending Stories
Grid of 3-4 trending stories based on engagement

### Statistics Spotlight
Key data points from recent stories

### Editor's Picks
Curated by editorial team

### Most Read
Based on page views

### Latest Stories
Reverse chronological, 6-8 stories

### Categories
Grid by section: Government, Economy, Technology, etc.

## Design

- Bloomberg-inspired data density
- Visual Capitalist-style infographics
- Dark theme with gold accents
- Responsive grid layout
- Lazy-loaded sections
- Smooth scroll navigation

## Input

Provide current story list with:
- slug
- title
- category
- summary
- hero image URL
- published date
- read time

## Output

Complete homepage HTML with all sections populated.
