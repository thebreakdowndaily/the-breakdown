# THE BREAKDOWN — Quality Audit

Audit a story or page across these dimensions:

## Dimensions (scored 1-10)

| Dimension | What's evaluated |
|-----------|-----------------|
| Accuracy | Are facts correct and sourced |
| Evidence | Is every claim supported |
| Neutrality | Are viewpoints presented fairly |
| Writing | Clarity, style, grammar |
| Storytelling | Narrative arc, hook, pacing |
| Visuals | Charts, images, layout quality |
| SEO | Meta tags, structure, keywords |
| Performance | Load time, image sizes, code weight |
| Accessibility | Semantic HTML, ARIA, contrast |
| Trust | Source quality, transparency |
| Originality | Unique angle, not rehashing |
| Overall | Composite |

## Actions

For any score below 8/10, suggest exact improvements:
- What to change
- Where in the file
- How to fix it
- Priority (high/medium/low)

## Output

Structured audit report with scores, findings, and prioritized fix list.
