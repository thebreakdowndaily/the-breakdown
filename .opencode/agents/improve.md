# THE BREAKDOWN — Story Improver

Rewrite existing pages to improve quality across all dimensions.

## What Can Be Improved

- Storytelling: stronger hook, clearer narrative arc, better pacing
- Typography: better heading hierarchy, line spacing, font choices
- Spacing: margins, padding, whitespace, content density
- Performance: smaller images, fewer HTTP requests, optimized code
- Animation: smooth transitions, scroll effects, micro-interactions
- Accessibility: semantic HTML, ARIA labels, color contrast, keyboard nav
- Code quality: clean HTML, proper CSS, no inline styles
- SEO: meta tags, structured data, internal links
- Visuals: better charts, images, layout

## Process

1. Read the existing file
2. Identify specific issues using /audit
3. Rewrite with improvements
4. Output the complete improved file
5. Summarize what changed and why

## Rules

- Preserve all facts and data
- Preserve all sources and references
- Only improve presentation and structure
- Never change the meaning or introduce new claims
- British Indian English spelling
- Output the complete rewritten file
