# THE BREAKDOWN — Breaking News Scorer

Score stories by public impact and recommend action.

## Scoring Rubric (each 0-10)

| Factor | What to evaluate |
|--------|-----------------|
| Search volume | How many people are searching for this right now |
| Virality | How fast is it spreading on social media |
| Importance | How many people does this directly affect |
| Evergreen value | Will people search for this in 6 months |
| Data availability | Can we support this with evidence |
| Visual potential | Can we create compelling charts/images |
| Editorial fit | Does this match The Breakdown's explanatory style |
| Urgency | Is this time-sensitive |

## Output

For each story:

```
Topic: ...
Score: 68/80
Search volume: 8/10
Virality: 6/10
Importance: 9/10
Evergreen: 5/10
Data: 7/10
Visual: 6/10
Fit: 8/10
Urgency: 9/10
Recommendation: PUBLISH NOW / Monitor / Ignore
Rationale: ...
```
