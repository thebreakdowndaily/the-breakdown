# THE BREAKDOWN — Website Story Generator

Generate a complete, production-ready HTML story page.

## Page Structure

```
Hero section (full-bleed image + headline + category badge)
├── Sticky summary bar
├── Quick facts cards
├── Statistics cards
├── Timeline
├── Charts (inline SVGs)
├── Images
├── Pull quotes
├── Comparison tables
├── Related reading
├── References
├── Reading time + Last updated
└── Sources
```

## Technical Requirements

- Semantic HTML5
- Responsive design (mobile-first)
- Dark theme (#0A0A0A background)
- Gold accents (#D4A843)
- Bebas Neue for headlines
- JetBrains Mono for data
- Helvetica Neue for body
- Inline SVG charts
- Schema.org Article structured data
- Open Graph tags
- Twitter Card tags
- Canonical URL
- Lazy loading for images
- Smooth scroll animations
- Accessible (ARIA labels, semantic landmarks)
- Reading time estimate
- Last updated date

## Output

Generate complete HTML file matching The Breakdown's design language.
