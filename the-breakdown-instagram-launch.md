## POST FORMATS

### 1. Carousel (Primary Format)
- **Dimensions:** 1080 x 1080 (square) or 1080 x 1350 (4:5 portrait)
- **Slides:** 7-10 slides
- **Style:** Dark background, white text, gold accents
- **Font:** Bebas Neue for headlines, Source Sans Pro for body
- **Structure:**
  - Slide 1: Hook (bold statement or question)
  - Slides 2-8: Analysis (one point per slide)
  - Slide 9: Key takeaway
  - Slide 10: CTA (follow, link in bio)

### 2. Reels (Secondary Format)
- **Duration:** 15-30 seconds
- **Dimensions:** 1080 x 1920 (9:16)
- **Style:** Fast cuts, text overlays, no host (graphics only)
- **Structure:**
  - 0-1s: Hook text (bold, centered)
  - 1-15s: Key points with visuals
  - 15-30s: CTA (follow for more)

### 3. Stories (Engagement Format)
- **Duration:** 15 seconds per story
- **Dimensions:** 1080 x 1920 (9:16)
- **Style:** Interactive (polls, questions, quizzes)
- **Use cases:** Updates, polls, behind-the-scenes, link drops

### 4. Static Posts (Occasional)
- **Dimensions:** 1080 x 1080 (square)
- **Style:** Quote cards, data visualizations, breaking news
- **Use cases:** Quick takes, breaking news, quotes

## VISUAL STYLE GUIDE (Instagram-Specific)

### Grid Layout
**Pattern:** Alternate between carousel and reel/static.
**Row pattern:** [Carousel] [Reel/Static] [Carousel] [Carousel] [Reel/Static] [Carousel]

This creates visual variety while maintaining consistency.

### Color Palette
- **Primary:** #1a1a2e (deep charcoal) — backgrounds
- **Secondary:** #ffffff (white) — text
- **Accent:** #e4a11d (gold) — highlights, emphasis
- **Danger:** #dc3545 (red) — negative outcomes, warnings
- **Neutral:** #888888 (warm gray) — secondary text

### Typography
- **Headlines:** Bebas Neue or Impact, all caps, tracked
- **Body:** Source Sans Pro or Inter, regular weight
- **Emphasis:** Bold or gold color (not both)
- **Minimum text size:** 24px (for mobile readability)

### Visual Elements
- **B icon mark:** Bottom-right corner, semi-transparent
- **Gold line:** Used as divider between sections
- **Dark backgrounds:** Always (ensures text readability)
- **No stock photos:** Use graphics, icons, and data visualizations only
