## 1. CAROUSEL TEMPLATE (Primary)

### Dimensions
- **Square:** 1080 x 1080px
- **Portrait:** 1080 x 1350px (4:5 — recommended for feed)
- **Slides:** 10 per carousel

### Slide Structure

#### Slide 1 (Hook)
```
Background: #1a1a2e (deep charcoal)
Layout: Center-aligned

[TOP] Small B icon mark (white on gold circle, 40px)
[CENTER] Bold headline text (Bebas Neue, 48-56px, white)
         One line. Maximum 8 words.
         Example: "INDIA BANNED TELEGRAM"
[BOTTOM] Subtext (Source Sans Pro, 18px, #888888)
         "10 things the headlines missed"
[VERY BOTTOM] Gold accent line (2px, centered, 120px wide)
```

#### Slides 2-8 (Analysis)
```
Background: #1a1a2e
Layout: Left-aligned

[TOP] Section number (Bebas Neue, 14px, gold #e4a11d)
      "THING #1" or "01" or topic name
[BELOW] Headline (Bebas Neue, 36px, white)
        "THE HUG IS GONE"
[BODY] 2-4 sentences (Source Sans Pro, 16-18px, #cccccc)
       One point per slide. Maximum 40 words.
[OPTIONAL] Visual element (icon, chart, comparison)
[BOTTOM] Thin gold line separator (1px, full width, 20% opacity)
```

#### Slide 9 (Key Takeaway)
```
Background: #1a1a2e
Layout: Center-aligned, large text

[CENTER] Key insight (Bebas Neue, 40px, white)
         "THE QUESTION ISN'T WHO HAS MORE LEVERAGE."
[BOTTOM] Context (Source Sans Pro, 14px, #888888)
         "It's whether both sides can use it without breaking the relationship."
```

#### Slide 10 (CTA)
```
Background: #1a1a2e
Layout: Center-aligned

[CENTER] B mark (large, 80px, gold outline)
[BELOW] "THE BREAKDOWN" (Bebas Neue, 24px, white, letter-spacing 3px)
[BELOW] "Complex stories. Clear analysis." (Source Sans Pro, 14px, #888888, italic)
[BELOW] "Follow for more breakdowns" (Source Sans Pro, 16px, gold)
```

### Canva Setup
1. Create custom size: 1080 x 1350
2. Set background: #1a1a2e
3. Add text boxes with specified fonts/sizes
4. Use gold (#e4a11d) for accents only
5. Export as PNG, 10 images
6. Upload as carousel

### Figma Setup
1. Frame: 1080 x 1350
2. Fill: #1a1a2e
3. Text styles: Create "Headline" (Bebas Neue 48), "Body" (Source Sans 16), "Caption" (Source Sans 14)
4. Component: Create reusable slide components
5. Export: 10 PNGs at 1x

## 2. REEL TEMPLATE (Secondary)

### Dimensions
- **Size:** 1080 x 1920 (9:16)
- **Duration:** 15-30 seconds
- **FPS:** 30

### Structure

#### 0-1s (Hook)
```
Background: #1a1a2e
Layout: Center-aligned, large text

[CENTER] Bold text (Bebas Neue, 64px, white)
         "INDIA BANNED TELEGRAM"
[BELOW] Subtext (Source Sans Pro, 24px, #888888)
        "in 48 hours"
[BELOW] Gold line animates in (2px, centered)
```

#### 1-15s (Content)
```
Background: #1a1a2e
Layout: Alternating center/left

Each point: 2-3 seconds
- Text fades in (Bebas Neue, 40px, white)
- Key stat or fact (Source Sans Pro, 20px, gold)
- Transition: slide up or fade

Maximum 5 points per Reel
```

#### 15-30s (CTA)
```
Background: #1a1a2e
Layout: Center-aligned

[CENTER] B mark (80px, gold outline)
[BELOW] "THE BREAKDOWN" (Bebas Neue, 28px, white)
[BELOW] "Follow for more" (Source Sans Pro, 18px, #888888)
```

### Canva Reel Setup
1. Create custom size: 1080 x 1920
2. Set background: #1a1a2e
3. Add text with animations (fade in, slide up)
4. Add timing: 2-3 seconds per text block
5. Export as MP4

### Editing Software
- **CapCut:** Free, easy text animations
- **DaVinci Resolve:** Free, professional
- **Premiere Pro:** Professional, paid

## 4. STATIC POST TEMPLATE (Occasional)

### Dimensions
- **Size:** 1080 x 1080 (square)

### Quote Card
```
Background: #1a1a2e
Layout: Center-aligned

[TOP] Opening quotation marks (gold, 72px)
[CENTER] Quote text (Source Sans Pro, 24px, white, italic)
         "The hug is gone. The hard bargaining has begun."
[BOTTOM] Attribution (Source Sans Pro, 14px, #888888)
         "The Breakdown, June 17 2026"
[VERY BOTTOM] B mark (small, centered)
```

### Data Card
```
Background: #1a1a2e
Layout: Left-aligned

[TOP] Label (Source Sans Pro, 14px, gold)
      "TELEGRAM USERS BY COUNTRY"
[CENTER] Bar chart or number visualization
         India: 250M (gold bar)
         Russia: 95M (gray bar)
         Iran: 83M (gray bar)
[BOTTOM] Source (Source Sans Pro, 12px, #888888)
         "Source: Statista, 2026"
```

### Breaking News Card
```
Background: #dc3545 (red)
Layout: Center-aligned

[TOP] "BREAKING" (Bebas Neue, 14px, white, letter-spacing 4px)
[CENTER] Headline (Bebas Neue, 40px, white)
         "US-IRAN PEACE DEAL SIGNED"
[BOTTOM] Time (Source Sans Pro, 14px, rgba(255,255,255,0.7))
         "June 16, 2026 — 2:47 PM IST"
```
