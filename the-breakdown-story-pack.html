Design Rules

  • ✓ Dark background on every story
  • ✓ Gold accent on every frame
  • ✓ B mark bottom-right (faded)
  • ✓ One message per story
  • ✓ Max 2 lines of text
  • ✓ No cluttered stickers

Typography

  • ✓ Titles: Bebas Neue, large
  • ✓ Body: Source Sans, 400 weight
  • ✓ Labels: Source Sans, uppercase
  • ✓ Max 3 font sizes per frame
  • ✓ Left-align or center
  • ✓ Never right-align

Tools

  • ✓ Canva (1080×1920 template)
  • ✓ CapCut (for animated stories)
  • ✓ Instagram native stickers
  • ✓ Use "Create" mode in Stories
  • ✓ Save templates for reuse
  • ✓ Batch-create 1 week at a time
